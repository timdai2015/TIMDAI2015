What is Wechatgogo - 微果

Wechatgogo is a startup that specializing in WeChat Official Account Wechat Website Design and development. WeChat is the most popular social media Asian Market.

微果是专注于为北美中小企业提供网络营销，IT顾问和软件设计开发等服务。自成立以来，与北美多家知名企业合作，并跟随互联网一同成长发展。

What did I contribute?

I was one of the UX/UI Designer in the team, my main contribution was to design the User Interface for the mobile users. I also helped them did the branding including Logo, typeface and color palette.

Mobile UI Design

The UI was designed to deliver best user experience to the end users as well as optimzed for programming. Over 50 UI designs were delivered, across 10 industries.The UI's are also Retina Display ready.

The Challenge

The biggest Challege is to find the sweet spot between deliving a pleasnt experiene to the users and making it not too difficult to code for the engineers on our team. Also, designing the system so it is easily scable for the future was a challenge.

Prototype

I also coded some of the designs in HTML , CSS and jQuery, so the back-end developers can focus on their strength.
